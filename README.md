# 📱 Deskripsi Aplikasi

CatatLari merupakan aplikasi mobile berbasis Flutter yang digunakan untuk mencatat aktivitas lari pengguna seperti tanggal lari, jarak tempuh, dan durasi lari.  

Aplikasi ini dikembangkan menggunakan arsitektur MVVM (Model View ViewModel) untuk memisahkan tampilan, data, dan logic aplikasi agar lebih terstruktur dan mudah dikembangkan.

---

# ✨ Fitur Aplikasi

## 🔐 Authentication
- Register akun
- Login akun
- Logout akun

## 🏃 Running Tracker
- Menambahkan data aktivitas lari
- Menampilkan daftar aktivitas lari
- Menghapus data aktivitas lari

## 👤 Profile
- Menampilkan data profile pengguna
- Update profile pengguna

## 💾 Local Storage
- Menyimpan data menggunakan SharedPreferences
- Data tetap tersimpan meskipun aplikasi ditutup

## 🧭 Navigation
- Bottom Navigation Bar
- Perpindahan halaman menggunakan Navigator Flutter
